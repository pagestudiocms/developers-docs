Facebook Tag¶

Sample code¶

{{ social:facebook_albums
    id="218908081677"
    breadcrumbs="n"
    cache="true"
    dimensions="75x65"
    albums="6"
    photos="6"
    modal="true"
    ul_class="flickr-list" }}
  {{ output }}
{{ /social:facebook_albums }}

Parameters¶

  • id=
  • breadcrumbs=
  • cache=
  • dimensions=
  • albums=
  • photos=
  • modal=
  • ul_class=

id=¶

id=

Specifies the Facebook page or profile id. Can be either numeric or alpha-numeric.

breadcrumbs=¶

breadcrumbs="n"

cache=¶

cache="true"

dimensions=¶

dimensions="75x65"

albums=¶

albums="6"

photos=¶

photos="6"

modal=¶

modal="true"

ul_class=¶

ul_class="flickr-list"
